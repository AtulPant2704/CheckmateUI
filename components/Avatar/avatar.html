Avatar

Avatar can be a user profile picture which can be used in social media or any other platform in form of a container to provide a visual identity to something.

Usage

Avatar in its simplest form inside a circular container containing a user profile picture.

Sizes

Add different sizes to an Avatar by adding avatar.lg, avatar.md or avatar.sm classes.
