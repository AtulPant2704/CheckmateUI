Image

An image is graphical representation of something to make a website much more appealing. Images are generally responsive which expands with the width of the container they are in.

Responsive Image

Make any image responsive by adding img-responsive class to it and fit it inside container of any width and it will expand by its container's width. 

                        
<img src="../images/image-res.jpg" alt="image-responsive" class="img-responsive">

Circle Image

Circle images are complete rounded images as well as responsive, you can use them by adding img-responsive and img-circle class to that img tag. 

                        
<img src="../images/avatar-1.jpg" alt="image-round" class="img-responsive img-circle">
