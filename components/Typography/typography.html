Documentation and text utilities which CheckMate follows that you can use in your project.

Paragraph

You can use different paragraph fonts for different font sizes like .body-cp-rg for regular font, .body-cp-lg for large font, .body-cp-md for medium font and .body-cp-sm for small font.

Alignment

You can use .align-left, .align-center and .align-right classes for aligning content to the left, center and right.

Extra Styles

  • To make a text gray, use .gray-text class.
  • To add strike-through on text, use .text-strike class.
  • To underline a text, use .text-underline class.
