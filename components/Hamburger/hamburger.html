Hamburger

Hamburger is present in the navbar and contains a list of items or routes to different pages which can be opened or closed on click of a button.

Simple Hamburger

This is a simple hamburger placed on top-left of screen and contains routes to different pages.(Go to mobile screen and click on hamburger to view menu items.)

//Javascript
const menuBarBtn = document.querySelector(".menu-bar");
const hamburgerMenu = document.querySelector(".hamburger-menu");
const menuBarClose = document.querySelector(".menu-bar-close");
 
menuBarBtn.addEventListener("click",openMenuBar);
menuBarClose.addEventListener("click",closeMenuBar);
 
function openMenuBar(){
   hamburgerMenu.classList.add("hamburger-menu-open");
}
 
function closeMenuBar(){
   hamburgerMenu.classList.remove("hamburger-menu-open");
}
