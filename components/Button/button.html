Button

Buttons are soul of the applications, they make an app much more interactive. They allow users to take actions and make choices with a single step.

Primary Button

Primary buttons are the most used buttons on a website, they will be of your primary color that you are having on your website. 

                        
<button class="btn btn-solid-primary">Solid</button>
<button class="btn btn-outline-primary">Outline</button>
<button class="btn btn-text-primary">Text</button>

Link Button

If you want to have a link with more emphasis then you can have a button with a link tag. 

                        
<button class="btn btn-link"><a href="#">Link</a></button>

Icon Button

Sometimes we want to improve the UX of a website or want to have more emphasis on a button then we can add icons to that button, as icons are visually more attractive as compared to plain text. 

                        
<button class="btn btn-text-icon btn-text-icon-primary"><i class="fas fa-shopping-cart"></i> Add to
    Cart</button>
<button class="btn btn-text-icon btn-text-icon-secondary"><i class="fas fa-paper-plane"></i>
    Send</button>
<button class="btn btn-text-icon btn-text-icon-primary"><i class="fas fa-trash-alt"></i> Delete</button>
<button class="btn btn-text-icon btn-text-icon-secondary"><i class="fas fa-save"></i>Save</button>

Floating Button

Icons can be used on a button to perform some toggle function or to perform a single function which can be easily interpreted by that icon. 

                        
<button class="btn btn-float btn-float-primary"><i class="fas fa-plus"></i></button>
<button class="btn btn-float btn-float-secondary"><i class="fas fa-arrow-up"></i></button>
<button class="btn btn-float btn-float-primary"><i class="fas fa-trash-alt"></i></button>
