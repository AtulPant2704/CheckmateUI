List

List is used in a website on navigation bar, stacked notifications, article pages, etc to show multiple lines of same information in an organised way.

Spaced List

Spaced List are the simple list contained inside a heading and contains different categories of that heading.

    Categories
  • Earphones
  • Laptop
  • Smartphones
  • Watches
                        
<ul class="list list-spaced">
    <div class="list-title">Categories</div>
    <li>Earphones</li>
    <li>Laptop</li>
    <li>Smartphones</li>
    <li>Watches</li>
</ul>

Stacked List

Stacked List are mostly used to show notifications in multiple lines.

  • Notification 1
  • Notification 2
  • Notification 3
  • Notification 4
